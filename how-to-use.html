How to use

Follow steps as shown below

    1. First off, a Metamask wallet is required to use the reservation channels. On Mobile, you will need to download the
    application and copy the reservation channel link to open it in the intergrated browser found in the
    Metamask mobile application. On PC, you will need to use the Metamask browser extesion.

    Metamask wesite:

    https://metamask.io/


    2. Using telegram, contact your choice of captain with the given information

    3. Go the captains reservation channels and select the one you were referred to. You can also use the
    Coinbase wei/thb converter along with the Footpath route planner to help you plan your trips

    4. Input the price + deposit to reserve.
    5. On the day of the trip, you have 3 options:


Cancel reservation - If trip is not confirmed, you can withdraw your funds (minus gas fees)

Approve trip - Sends agreed price to captain, Deposit gets sent back to you

Refund - If trip is confirmed, you can request a refund (captain must confirm refund)
